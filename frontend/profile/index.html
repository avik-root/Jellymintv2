Querying the API via Command Line

Use your bearer token to hit the endpoint directly using standard request tools like curl:

curl -X POST "https://spousal-scrabble-stamina.ngrok-free.dev/api/chat" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_TOKEN" \
  -d '{
    "model": "qwen",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
